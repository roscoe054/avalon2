模拟alert（type="alert"）

说明：当配置type为alert时，showClose的配置无效，也就是其为false还是true都不会显示“关闭”按钮

模拟alert 呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵 
                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>模拟alert</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset>  
                            <legend>模拟alert</legend>
                                <button ms-click="show('bb')">打开对话框</button>
                                <oni:dialog $id="bb" config="opts" $slot="content">
                                    设置了type之后，无论showClose为true or false都不显示关闭Icon
                                </oni:dialog>
                        </fieldset>
                        <script> 
                            require(["./component/dialog/avalon.dialog"], function() {
                                avalon.define({
                                    $id: "demo",
                                    $skipArray: ["opts"],
                                    opts: {
                                        type: 'alert',
                                        title: '警告框demo',
                                        showClose: true
                                    },
                                    show: function (id) {
                                        avalon.vmodels[id].toggle = true;
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
