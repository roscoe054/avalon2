模拟alert（type="alert"）

说明：通过将dialog配置对象的type属性设为alert，将模拟alert的表现，也就是说不会存在“关闭”、“取消”按钮

模拟alert 呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵 
                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>模拟alert</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset>  
                            <legend>模拟alert</legend>
                                <button ms-click="show('bb')">打开对话框</button>
                                <oni:dialog $id="bb" config="opts" $slot="content">
                                    呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵
                                </oni:dialog>
                        </fieldset>
                        <script> 
                            require(["./component/dialog/avalon.dialog"], function() {
                                avalon.define({
                                    $id: "demo",
                                    $skipArray: ["opts"],
                                    opts: {
                                        type: 'alert',
                                        title: '警告框demo'
                                    },
                                    show: function (id) {
                                        avalon.vmodels[id].toggle = true;
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
